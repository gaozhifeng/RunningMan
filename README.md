# 关于：
> **RunningMan** 多进程Socket 框架

    1.start stop restart reload status 指令
    2.SIGINT(stop) SIGUSR1(reload) SIGUSR2(status) 信号
    3.TCP UDP 网络协议支持
    4.select poll epoll 事件处理模型支持
    5.Text PB Json 数据序列化支持
